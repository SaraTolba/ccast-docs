# Acronyms

Terms used throughout these guides. Abbreviations defined in
`includes/abbreviations.md` also appear as tooltips wherever they occur in the
docs, so hovering over an acronym on any page will show its expansion.

| Acronym | Expansion |
| --- | --- |
| ACL | Access Control List |
| CCAST | Center for Computationally Assisted Science and Technology |
| CPU | Central Processing Unit |
| GPU | Graphics Processing Unit |
| HPC | High-Performance Computing |
| MKL | Math Kernel Library |
| MPI | Message Passing Interface |
| NDSU | North Dakota State University |
| PBS | Portable Batch System |
| Slurm | Simple Linux Utility for Resource Management |
| SSH | Secure Shell |
