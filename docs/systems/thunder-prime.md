# Thunder Prime Cluster

*Thunder Prime Cluster hardware specifications.*

The Thunder Prime cluster (often referred to as “Prime”) is comprised of 87 AMD EPYC nodes in 64-, 128-, and 192-core configurations. In total,
Prime contains 9,600 CPU cores, 46 TB aggregate memory (RAM), 99 Nvidia GPUs, and a 2 PB GPFS parallel filesystem.
