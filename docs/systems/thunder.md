# ThunderCluster

*Thunder Cluster hardware specifications*

The Thunder cluster is comprised of 117 nodes containing multiple generations of Intel CPUs. In total, Thunder contains 3,632 CPU cores, 16 TB
aggregate memory (RAM), and 14 Nvidia GPUs. Thunder shares a 2.2 PB parallel filesystem with the Thunder Prime cluster. This cluster is
currently in attrition mode.
